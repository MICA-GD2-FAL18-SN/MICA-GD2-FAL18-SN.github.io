In-class Assignment | GD 220-01 | Fall 2018 | 3 CREDITS

Poster Sites

Deliverables
HTML / CSS webpage

Process

  1. Create a github repository with the following name: [your username].github.io
  2. Think of two Contrasting words (A&B) that represent yourself in some way— your Interests, personality traits, your design sensibility.
  3. List 10 other words that relate to both A and B
  4. Write or find a phrase or quote for each of your original words
  5. Choose two visual elements (they could be created by you or from the web) that relate to each of your words
  6. Create an HTML document in your repository folder that includes all of these elements
  7. Using these different items, use position, transform, and animation to create a fluid, moving "poster". See Buhoy Kim's Studio Odd Hyphen for inspiration.
